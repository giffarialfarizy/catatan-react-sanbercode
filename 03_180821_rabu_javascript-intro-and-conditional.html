Catatan Ringkas

  1. Menggabungkan dengan concat()

    var string1 = 'good';

    var string2 = 'luck';

    console.log(string1.concat(string2)); // goodluck

  2. Memilih karakter ke-n dengan charAt()

    console.log('i am a string'.charAt(3)); // 'm'

  3. Memilih karakter antara karakter ke-x da ke-y dengan substr(x, y)

    var car1 = 'Lykan Hypersport';

    var car2 = car1.substr(6);

    console.log(car2); // Hypersport

  4. Ternary Operator merupakan pengondisian yang bisa di buat dengan satu baris

    var age= 20

    var bisaVote= age > 20 ? "bisa vote" : "belum bisa vote" // hasilnya belum bisa vote

    var angka = 9

    var jenisBilangan = angka % 2 === 0 ? "Bilangan Genap" : "Bukan Bilangan Genap" //hasilnya bukan bilangan genap

Kata kunci

  • concat()
