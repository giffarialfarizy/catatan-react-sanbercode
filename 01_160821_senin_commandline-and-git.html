Catatan Ringkas

  1. Command Line Interface: membuat file baru

    $ touch index.html

    untuk membuat file baru yaitu index.html

  2. Rekomendasi terminal: cmder

  3. Perintah dasar git

    1. git init
    2. git add
    3. git commit
    4. git status
    5. git push
    6. git pull

  4. Mendorong folder yang sudah ada untuk menjadikannya remote repository

    1. cd demo-git

    2. git init

    3. git remote add origin https://github.com/yogimahaputra/demo-git.git

    4. git add .

    5. git commit -m "Initial commit"

    6. git push origin master

  5. Untuk clone Remote Repository menggunakan SSH

    1. https://www.petanikode.com/github-ssh/
    2. https://rakitaplikasi.com/blog/cara-menggunakan-ssh-di-github
    3. https://docs.github.com/en/github/authenticating-to-github/connecting-to-github-with-ssh

Kata kunci

  • Membuat file baru
  • Rekomendasi terminal
  • Perintah dasar git
  • Membuat remote repository dari folder yang sudah ada
